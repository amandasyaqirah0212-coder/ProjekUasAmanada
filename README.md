3. Fitur Utama
   . Firebase Integration: Sinkronisasi data menu, harga, dan pesanan secara real-time dari Firebase.
   . Horizontal Scrolling: Implementasi RecyclerView untuk menampilkan promo terbaru, rekomendasi kopi best-seller, dan kategori menu.
   . Detail Menu: Menampilkan deskripsi varian kopi, pilihan customization dan menggunakan Glide untuk memuat gambar produk resolusi tinggi.
   . SharedPreferences: Fitur preferensi pengguna untuk menyimpan riwayat pesanan terakhir dan lokasi yang tersimpan.
   . Admin Dashboard: Panel khusus untuk mengelola stok bahan, mengubah status pesanan, serta menambah menu musiman (seasonal drinks).
   . Filter Category: Memudahkan pengguna memfilter minuman berdasarkan kategori seperti Essential Coffee, Non-Coffee, Ice Shaken.
